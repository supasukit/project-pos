# 🧾 Project POS (Electron + SQLite)

ระบบ Point of Sale (POS) สำหรับร้านค้าขนาดเล็ก  
พัฒนาโดยใช้ **Electron.js**, **SQLite**, และ **Vanilla JavaScript**

## 📦 ฟีเจอร์หลัก

- จัดการสินค้า (เพิ่ม/ลบ/แก้ไข)
- จัดการลูกค้า และยอดค้างชำระ
- ระบบชำระเงินและประวัติการขาย
- ใช้งานได้แบบ Offline
- เชื่อมต่อฐานข้อมูล SQLite
- รองรับ Barcode Scanner

---

## 🚀 วิธีใช้งาน

```bash
git clone https://github.com/supasukit/project-pos.git
cd project-pos
npm install
npm start
```

> หากยังไม่มี Electron:
```bash
npm install electron --save-dev
```
